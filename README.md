\# CryptoWatcher



Telegram-бот для мониторинга криптовалютных цен с алертами.



\## Что умеет



Бот поддерживает два источника котировок:



\- \*\*Bybit\*\*

\- \*\*CoinMarketCap (CMC)\*\*



Для каждого источника доступны два режима:



\- \*\*Top-N\*\*

\- \*\*заранее заданный список монет / пар\*\*



Вся логика настраивается через `.env`, без правки кода.



\---



\## Логика работы



\### Источник котировок



Переменная `PRICER` определяет, откуда брать цены:



\- `BYBIT`

\- `CMC`



\### Основной триггер алерта



`THRESHOLD\_PERCENT` — это \*\*порог изменения от сохраненной базовой цены\*\*.



Примеры:



\- `20`

\- `15`

\- `5`

\- `1.2`



Важно:



\- это изменение \*\*от базы\*\*

\- это \*\*не\*\* изменение за `POLL\_INTERVAL`

\- это \*\*не\*\* изменение за `CHANGE\_TF`



\### Частота опроса



`POLL\_INTERVAL` — как часто бот проверяет цены.



Примеры:



\- `5m`

\- `10m`

\- `15m`

\- `30m`

\- `60m`



\### Дополнительная метрика



`CHANGE\_TF` — это \*\*информационный таймфрейм\*\* для дополнительного процента по свечам.



Он используется только для отображения в алертах и `/status`, но \*\*не запускает алерт сам по себе\*\*.
