# CryptoWatcher

Telegram-бот для мониторинга криптовалютных цен с алертами.

## Что умеет

CryptoWatcher отслеживает цены и присылает алерты в Telegram, когда цена уходит от сохраненной базовой точки на заданный процент.

Поддерживается:

- **Bybit**
- **CoinMarketCap (CMC)**

Для каждого источника доступны два режима:

- **Top-N**
- **свой список монет / пар**

Также бот умеет:

- хранить настройки **отдельно для каждого чата**
- менять настройки прямо в Telegram через **inline-кнопки**
- показывать `/status`
- временно отключать алерты по монете через `mute`
- сбрасывать baseline через `reset base`
- работать в Docker как фоновый сервис
- сохранять состояние между перезапусками

## Как устроена логика алертов

### Основной триггер

`THRESHOLD_PERCENT` — это **порог изменения от сохраненной базовой цены**.

Примеры:

- `20`
- `15`
- `5`
- `1.2`

Важно:

- это изменение **от базы**
- это **не** изменение за `POLL_INTERVAL`
- это **не** изменение за `CHANGE_TF`

### Частота проверки

`POLL_INTERVAL` — как часто бот должен проверять рынок для конкретного чата.

Примеры:

- `5m`
- `10m`
- `15m`
- `30m`
- `60m`

### Дополнительная TF-метрика

`CHANGE_TF` — это дополнительная информационная метрика по свечам.

Она:

- показывается в `/status` и алертах
- **не является триггером**
- используется только как дополнительный контекст
